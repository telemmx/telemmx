## 🔥2. Batch create channels/groups
Create channels or groups in batches, set administrators, and send a message to the top

## 🔥3. Collect users V1&V2
It can collect nearby people, groups/channels, contacts, chatted people, private groups, etc. The collection is accurate, and it can collect large groups and hidden groups of members.

You can directly pull users, or you can pull them directly after collecting them from a group.

You can directly send users privately, or you can collect them directly from groups and send them privately. At the same time, private posting supports fake news, pinning, etc., to notify users and increase exposure.

Can automatically reply to key users in the group or information containing keywords, and can forward or directly send information to other groups

## 🔥10. Automatic private chat
Listen to the users who send you messages or keywords of messages, and reply, which is equivalent to automatic customer service.
