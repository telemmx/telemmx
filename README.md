# TELEONE
什麼是TELEONE：TELEONE是一個集成多賬號，支持發送md/html消息，群組用戶採集，批量拉人，自動群聊，群發建群，批量註冊，刷消息閱讀量，養號，賬號篩活，並支持動態代理的TG(telgrame)網絡營銷工具。

![軟件主界面](https://github.com/teleone-tech/teleone/blob/main/home.jpg?raw=true)

## 🔥1. 註冊/登陸
  手動登陸註冊賬號，導入賬號，批量修改賬號，批量退群，代理切換，接收官方信息等
## 🔥2. 批量創建頻道/群組
   批量創建頻道或群組，設置管理員，並可發送一條消息然後置頂
## 🔥3. 採集用戶 V1&V2
   可採集附近的人，群組/頻道，聯繫人，聊過的人，私有群組等。採集精準，可採集大群及成員隱藏群組。
## 🔥4. 採集群組
   可採集附近的群，電報默認搜索，機器人搜索引擎，網絡搜索等。
## 🔥5. 批量拉人 V1&V2&V3
   可直接拉用戶，也可以從群組採集後直接拉。
## 🔥6. 批量私發 V1&V2
   可以直接私髮用戶，也可以直接從群組採集後私發。同時私發支持偽消息、置頂等，給用戶通知，提高曝光率。
## 🔥7. 自動群聊（炒群）
   自動聊天炒群活躍氣氛，也可以作為廣告群發。支持添加話術，回复消息等。
## 🔥8. 自動群發 V1&V2
   可設置時間間隔群發，也可以設置間隔消息數量群發。支持偽消息。
## 🔥9. 自動群回复/轉發
   可自動回复群內關鍵用戶或包含關鍵字的信息，同時可以轉發或直發信息到其他群組
## 🔥10. 自動私聊
   監聽給你發信息的用戶或信息的關鍵字，進行回复，相當於自動客服。
## 🔥11. 自動營銷
   可以自動群發，自動回复，自動私聊，死號自動新號抵上，適合挂機。
## 🔥12. 刷閱讀量
   刷頻道或群組某個信息的閱讀量，烘托氣氛提高人氣
## 🔥13. 同步群組
   可將群組或頻道內的圖片視頻或全部內容等下載到本地，也可以轉發或直發到其他群組，並可加上自己的信息。
## 🔥14. 號碼篩活 V1&V2
   可根據號段產生號碼，並可判斷號碼是否註冊，是否封禁，是否可用
## 🔥15. 批量註冊
   對接接碼平台，採用官方API自動註冊，可設置二次密碼。
## 🔥16. 批量養號
   新註冊號碼可適當在群內發言，聊天，保持活躍度。
   
 * ❗️本軟件支持普通文本，Markdown，HTML
 * ❗️本軟件除支持用戶名，也支持手機號
 * ❗️本軟件支持私有群組
 * ❗️本軟件支持發送文字，圖片、視頻及語音
 * ❗️本軟件支持最新URL格式
